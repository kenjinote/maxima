Information about included software

This installer for Maxima does not contain only Maxima, but also other free software, which is required for Maxima. The following projects are included in the installer:

  • Maxima Website, Maxima license
  • CLISP Website, CLISP license
  • SBCL Website, SBCL license
  • ABCL Website, ABCL license
  • TCL/TK Website, TCL/TK license
  • Gnuplot Website, Gnuplot license
  • VTK Website, VTK license
  • wxWidgets Website, wxWidgets license
  • wxMaxima Website, wxMaxima license
Thanks to all these free software projects! And the free software which was used to build this installer!
